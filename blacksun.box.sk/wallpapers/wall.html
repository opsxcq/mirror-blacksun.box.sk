Blacksun Wallpapers

Note: These are the wallpapers that our fans have created for your usage. If you are a die hard fan I bet you will want to place these on your box. If you would like to contribute wallpapers to this collection, please email Mikkkeee the wallpaper and it will find its place on this page.

Installing


For all users, depending on how you want to download the images, we would like you to convert the .jpg images to .bmp in order for your os to use them. If you don't know how to do this, simply install irfanview, a freeware image manipulator program. Open up the image then save it as a .bmp, yup simple as that. For unix / linux users i won't bother, if you know how to install linux then you should be able to change your wallpaper.
